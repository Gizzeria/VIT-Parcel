About VIT Parcel

VIT Parcel is a community-based parcel pickup coordination system created for hostel students.

We realized that many students spend a lot of time walking to the main gate to collect their parcels. Often, someone from the exact same block is already going there but has no way to coordinate and help others. VIT Parcel bridges this gap.

Our Mission

To save time and foster a sense of community among hostel mates by allowing seamless coordination for everyday tasks.

Technologies Used

  • HTML5 Structure (nav, article, aside, section)
  • CSS3 (Flexbox/Grid layout, colors, variables)
  • Vanilla JavaScript (DOM Manipulation, Events, While/Do loops)
  • LocalStorage API for Data Persistence
